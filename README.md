# Data Cleaning with SQL

## Project Overview
This project demonstrates how raw data can be cleaned and transformed using SQL. It focuses on identifying inconsistencies, handling missing values, and preparing data for analysis.

## Objectives
- Remove duplicate records
- Standardize inconsistent text values
- Handle missing data
- Prepare a clean dataset for analysis

## Tools Used
- SQL
- CSV Data

## Data Issues Found
- Duplicate records
- Inconsistent text formatting
- Missing values

## Cleaning Steps
- Removed duplicates
- Standardized city and department names
- Replaced NULL values
- Validated final dataset

## Files Included
- raw_data.csv
- cleaned_data.csv
- cleaning_queries.sql
